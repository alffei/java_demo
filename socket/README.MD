### Socket与TCP、UDP

What is Socket?
- 简单来说，Socket是IP地址和端口结合的协议（Socket = IP Addr + Port Num）
- 在网络传输中用于唯一标识两个端口（IP + Port）之间的链接

### Socket之TCP
- TCP是面向连接的通信协议
- TCP是端到端的链接（P2P）

### Socket之UDP
- TCP是面向无连接的通信协议
- UDP数据包含目的端口号和源端口号信息
- 可以实现广播，不限于端到端

### 报文段
 - 充当TCP/IP协议的路由功能
 - 报文在传输过程中会不断地封装成分组、包、帧来传输
 
### 传输协议
- 一种规定，约束

### IP地址
- 1.1.1.1 直接广播地址
- 255.255.255.255 首先广播地址
- ipv6 128位（64位网络前缀 + 64位主机地址）
- ipv4一定可以转为ipv6,但ipv6不一定能转为ipv4

### 端口
49152到65535是动态端口号

### UDP单播、广播、多/组播
- 受限多播地址：255.255.255.255
- IP:157.0.134.94
- 子网掩码:255.255.0.0
